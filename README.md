Split Ball is a lightweight browser-based physics game built using HTML5 Canvas and vanilla JavaScript. Balls move inside a circular arena, collide with moving pipes, and split into additional balls with visual and audio feedback.

## Features

- Circular arena with boundary collision
- Two vertically moving pipes inside the arena
- Ball splitting on pipe collision with cooldown
- Particle explosion and sound effect on split
- Heads-up display showing ball count and split count
- Reset game functionality
- Responsive canvas resizing
- Ball limit (1500)

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Canvas API
- Web Audio API

No external libraries or frameworks are used.

## Project Structure
- index.html
- style.css
- script.js

## How to Run

1. Clone the repository
```js
git clone https://github.com/coding-blocks/BallGame
```

2. Open index.html in any modern browser

No build steps or server setup required.

## Controls

- Mouse click: Spawn a new ball
- Reset button: Restart the game
